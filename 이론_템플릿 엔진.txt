템플릿 엔진
    지정된 템플릿 양식과 데이터가 합쳐져 HTML 문서를 출력하는 소프트웨어를 이야기한다
    대표적으로 JSP, Freemarker 그리고 React, Vue 등이 있다
    서버 템플릿 엔진 : JSP, Freemarket
    클라이언트 템플릿 엔진 : React, Vue

※ JSP는 명확하게는 서버 템플릿 엔진은 아니지만, View의 역할만 하도록 구성할 때는 템플릿 엔진으로써 사용 가능 (이 경우엔 Spring + JSP 로 사용한 경우)


서버 템플릿 엔진
   서버 템플릿 엔진은 자바 스크립트가 구동하는 프론트가 아닌 서버(백)에서 작동한다
   서버 템플릿 엔진을 이용한 화면 생성은 서버에서 Java 코드로 문자열으 만든 뒤 이 문자열을 HTML로 변환하여 브라우저로 전달한다


클라이언트 템플릿 엔진
    브라우저 위에서 작동한다
    서버 템플릿 엔진의 손을 벗어낫기 때문에 제어할 수가 없다
    Vue.js 혹은 React.js 를 이용한 SPA(Single Page Application)는 브라우저에서 화면을 생성한다 (즉, 서버에서 이미 코드가 벗어났다)
    서버에서는 Json 혹은 Xml 형식의 데이터만 전달하고 클라이언트에서 조립한다
